# COGA Electron Icons

This directory contains icons for the COGA desktop application.

For production, you should replace these placeholder files with proper icons:

- `icon.png` - Main icon (256x256 recommended)
- `icon.ico` - Windows icon format
- `icon.icns` - macOS icon format

## Creating Icons

You can use tools like:
- ImageMagick to convert PNG to ICO
- GIMP or Photoshop for creating icons
- Online icon generators

## Current Status

Using placeholder structure for development. Icons will be loaded when the app starts.
